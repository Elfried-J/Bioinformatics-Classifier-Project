# DNA Species Classifier

A machine learning app that classifies DNA sequences across 6 species using k-mer frequency analysis.

## Species
- Escherichia coli
- Saccharomyces cerevisiae
- Drosophila melanogaster
- Homo sapiens
- Mycobacterium tuberculosis
- Arabidopsis thaliana

## Models
- SVM (97% accuracy)
- Random Forest (92% accuracy)
- Logistic Regression (95% accuracy)
- 1D CNN (96% accuracy)
